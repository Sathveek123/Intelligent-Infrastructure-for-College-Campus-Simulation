# Quickstart

## Prerequisites

- Node.js 18+
- npm 9+

## Run Frontend

From `frontend/`:

```bash
npm install
npm run dev
```

Open the printed local URL in your browser.

## Production Build

From `frontend/`:

```bash
npm run build
npm run preview
```

- **Dashboard**

  - Advanced Systems (Phases 9–12)
